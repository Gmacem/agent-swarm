## Instructions

1. Check `prompts/project.md` for the linter and config.
2. Run the linter on the specified target (or the whole project).
3. Report results:
   - Total issues found (errors vs warnings).
   - For each issue: file, line, rule, message.
4. If there are auto-fixable issues, fix them.
5. Report what was fixed and what needs manual attention.
