# Update Task

Report progress, blockers, or completion on your current task.

## Arguments

- `$ARGUMENTS` â€” Free-text status update (e.g. "completed auth module" or "blocked on API spec")

## Instructions

1. Find your active task in `tasks/active/`.
2. Update the task file:
   - Update the **Status** field if it changed (in-progress, blocked, review, done).
   - Update the **Updated** date to today.
   - Add a timestamped note under **Notes** with your update.
3. If the status is `blocked`, describe the blocker and which role can unblock it.
4. If the status is `done`, check off completed acceptance criteria and summarize what was delivered.
5. Confirm the update was made.
